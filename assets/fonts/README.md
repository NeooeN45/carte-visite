# Dossier polices

Vide par défaut : Manrope et Inter sont chargées depuis Google Fonts (voir
`index.html`). Si vous préférez un hébergement 100 % local, placez ici les
fichiers `.woff2` de ces polices et ajoutez les règles `@font-face`
correspondantes en haut de `assets/css/style.css` — voir le README à la
racine du projet, section « Polices : CDN vs auto-hébergement ».
